# 🎧 SonicBeast IEM Guide

A modern and responsive **In-Ear Monitor (IEM) learning & recommendation website** built using **HTML, CSS, and JavaScript**.  
This project helps beginners understand IEM technology, sound signatures, frequency graphs, and choose the right audio gear through an interactive experience.

---

## 🚀 Features

### 🎵 IEM Learning Platform
- Explains different IEM driver technologies:
  - Dynamic Driver (DD)
  - Balanced Armature (BA)
  - Planar Magnetic Driver
  - Piezoelectric Driver

### 📊 Interactive Sound Graphs
- Visual explanation of frequency response:
  - Bass
  - Mids
  - Treble
- Different tuning styles:
  - Bass Heavy
  - Balanced
  - Bright Details

### 🎧 IEM Comparison Section
Compare popular IEM models based on:
- Bass response
- Vocal clarity
- Gaming accuracy
- Micro details

Featured models:
- KZ EDX Pro
- Simgot EW300
- Truthear Hexa

### 🧠 Smart IEM Selector
Interactive recommendation tool that suggests an IEM based on:
- Budget
- Usage preference
- Sound preference

### 💻 Modern UI/UX
- Responsive design
- Glassmorphism style cards
- Gradient effects
- Smooth animations
- Mobile navigation menu
- Hover interactions
